Politique de Confidentialité

Dernière mise à jour :

1. Introduction

Le Délice Burger ("nous", "notre", "nos") exploite l'application mobile Le Délice Burger (le "Service").

Cette page vous informe de nos politiques concernant la collecte, l'utilisation et la divulgation des données personnelles lorsque vous utilisez notre Service.

2. Données Collectées

2.1 Données de compte

  • Nom et prénom
  • Numéro de téléphone
  • Adresse email
  • Adresse de livraison

2.2 Données de localisation

Nous collectons votre position GPS uniquement lorsque vous passez une commande pour :

  • Définir votre adresse de livraison
  • Calculer les frais de livraison
  • Optimiser le trajet du livreur

2.3 Données de commande

  • Historique de vos commandes
  • Préférences alimentaires
  • Méthode de paiement choisie

3. Utilisation des Données

Nous utilisons vos données pour :

  • Traiter et livrer vos commandes
  • Vous contacter concernant votre commande
  • Améliorer notre service
  • Envoyer des promotions (avec votre consentement)

4. Partage des Données

Nous ne partageons vos données qu'avec :

  • Nos livreurs : nom, téléphone et adresse de livraison uniquement pendant la livraison active
  • Prestataires de paiement : pour traiter vos paiements de manière sécurisée

Nous ne vendons JAMAIS vos données personnelles à des tiers.

5. Sécurité des Données

Nous utilisons des mesures de sécurité techniques et organisationnelles pour protéger vos données :

  • Chiffrement SSL/TLS pour toutes les communications
  • Stockage sécurisé dans des bases de données protégées
  • Accès limité aux employés autorisés uniquement

6. Vos Droits (RGPD)

Conformément au RGPD, vous avez le droit de :

  • Accéder à vos données personnelles
  • Rectifier vos données incorrectes
  • Supprimer votre compte et vos données
  • Exporter vos données dans un format lisible
  • Retirer votre consentement à tout moment

7. Suppression de Compte

Pour supprimer votre compte et toutes vos données :

  1. Ouvrez l'application Le Délice Burger
  2. Allez dans Profil > Paramètres
  3. Cliquez sur "Supprimer mon compte"
  4. Ou envoyez un email à : support@ledeliceburger.cm

Vos données seront supprimées sous 30 jours.

8. Conservation des Données

  • Données de compte : conservées tant que votre compte est actif
  • Historique de commandes : 3 ans (obligation légale comptable)
  • Données de localisation : supprimées après livraison

9. Cookies

L'application utilise des cookies techniques essentiels pour :

  • Maintenir votre session connectée
  • Mémoriser votre panier

10. Modifications

Nous pouvons modifier cette politique de confidentialité. Nous vous notifierons de tout changement important via l'application.
